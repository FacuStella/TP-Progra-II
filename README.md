# 🏕️ Sistema de Gestión de Guardería de Motorhomes

## 📌 Descripción
Este proyecto implementa un sistema de gestión para una **guardería de motorhomes**, permitiendo administrar de forma organizada:
- Socios y usuarios
- Vehículos registrados
- Garages y zonas
- Empleados asignados
- Servicios de mantenimiento y control

El objetivo es ofrecer una solución **modular, escalable y mantenible** que facilite la administración de la guardería, optimizando la experiencia tanto para los socios como para el personal.

---

## ⚙️ Funcionalidades principales
- 📋 **Registro y consulta de vehículos**  
- 🏢 **Gestión de garages y zonas**  
- 👥 **Administración de socios y empleados**  
- 🔧 **Control de mantenimiento contratado**  
- 💡 **Lectura de contador de luz por garage**  
- 🔐 **Sistema de usuarios con menú interactivo**

---

## 🛠️ Tecnologías utilizadas
- **Lenguaje:** Java ☕  
- **Paradigma:** Programación Orientada a Objetos (POO)  
- **Principios:** Modularidad, escalabilidad y reutilización de código  
- **Herramientas:**  
  - Clases y objetos para modelar entidades  
  - Catálogos centralizados para instanciación dinámica  
  - Utilidades para inicialización y persistencia de datos
